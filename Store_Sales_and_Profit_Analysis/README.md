# Intermediate Level Task 01 - Store Sales and Profit Analysis

This project analyzes store sales and profit using the *Sample_Superstore.csv* dataset. It identifies key business insights such as top-performing categories, profitable regions, and sub-category trends.

## Dataset
- File: **Sample_Superstore.csv**
- Rows: 9994
- Columns: 21
- Contains order details, sales, profit, category, region, and customer information.

## Analysis Performed
- Data loading and summary
- Missing value check
- Total sales and profit calculation
- Average sales and profit
- Sales by Category
- Profit by Category
- Sales by Sub-Category
- Profit by Sub-Category
- Profit by Region
- Visualizations using bar charts

## Key Findings
- Technology category generated the highest sales.
- Office Supplies and Technology produced the most profit.
- Some sub-categories are highly profitable while others show losses.
- Regional analysis helps identify strong and weak markets.

## How to Run
Install required libraries:
